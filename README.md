# 🎬 Netflix Clone

Um projeto front‑end inspirado na interface da Netflix, desenvolvido para praticar HTML, CSS e JavaScript.  
Inclui um **banner principal** e um **carrossel de filmes/séries** implementado com [Swiper.js](https://swiperjs.com).

---

## 🚀 Tecnologias utilizadas
- **HTML5** para estrutura da página  
- **CSS3** para estilização e responsividade  
- **JavaScript (ES6)** para interatividade  
- **Swiper.js** para o carrossel responsivo  
- **Font Awesome** para ícones  

---

### Estrutura do Projeto

net_clone/    
├── index.html  
├── assets/        # imagens (capa, miniaturas)  
├── style/         # estilos principais  
│   ├── main.css  
│   └── responsive.css   



---

## ⚙️ Como rodar
1. Clone este repositório ou baixe os arquivos.  
2. Abra o arquivo `index.html` em qualquer navegador moderno.  
3. Certifique-se de estar conectado à internet para carregar o **Swiper.js** e o **Font Awesome** via CDN.  

---

## 🎨 Funcionalidades
- Banner inicial com título e descrição.  
- Carrossel horizontal de filmes/séries.  
- Responsividade (1 item em celular, 3 em tablet, 5 em desktop).  
- Efeito de hover nas capas (zoom leve).  
- Navegação manual via setas e paginação.
